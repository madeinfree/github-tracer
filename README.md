# Github Tracer
Simply for you to trace your github repo, used github api, jQuery and React.

![Github-tracer](http://i.imgur.com/AAGo1Cg.png)

# Installation

```
git clone https://github.com/madeinfree/github-tracer.git
```

Changed the user in

```javascript
<script type='text/babel'>
  var user = 'your name';
</script>
```

and `open index.html`

Well done!
